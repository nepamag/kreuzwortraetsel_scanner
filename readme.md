# Kreuzworträtsel Scanner

Die App "Kreuzworträtsel Scanner" ist eine App, die lediglich dazu entwickelt wurde, die Kreuzworträtsel des Tagesanzeiger Magazins Trudy Müller-Bosshard
einscannen zu können, um dann die einzelne Antworten separat angezeigt zu erhalten, ohne die restlichen Lösungen zu sehen.

## Benutzung

1. Klick auf "Bild auswählen" um ein gespeichertes Bild zu laden, oder auf "Foto aufnehmen" um ein Foto aufzunehmen.
2. Drehen des Bilds sodass der Text waagrecht ist (keine Vorschau vorhanden, muss nach Erinnerung erfolgen)
3. Auf "Scan" klicken, und überprüfen, dass das Bild richtig gedreht wurde, ansonsten den Ablauf von vorne beginnen
4. Nach Abschluss kann durch Klick auf "Waagrecht" oder "Senkrecht" und Eintippen der Nummer die entsprechende Lösung gewählt werden.
